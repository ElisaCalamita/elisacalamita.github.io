I made this video in response to the assignment of an illustration class. The assignment was to illustrate a minor event based on a childhood memory that was horrifying, but at the same time very funny. The memory I chose the illustrate, is an event which I will never forget. You can see that a memory like this that happened so long ago, is really hard to forget.

 

In this video I experimented telling a narrative in images by making the combination of illustration and video  projection.

De herinnering die ik hieronder heb uitgewerkt, is mij altijd bijgebleven. Zo zie je maar weer dat iets dat zo lang geleden is gebeurd, toch moeilijk kan worden vergeten.
